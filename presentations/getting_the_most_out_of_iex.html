Debug

IEx.pry 

defmodule User do
  def hello(name) do
    require IEx; IEx.pry

    "Hello #{name}!"
  end
end

When the function is called, drop in at the point of IEx.pry 

iex> User.hello("Jim")
...
pry(1)> name
"Jim"

Remote shells

Naming a shell

$ iex --sname foo
Erlang/OTP 19 ...

Interactive Elixir ...
iex(foo@hostname)1>

Connecting to a remote shell

$ iex --sname bar --remsh foo@hostname
Erlang/OTP 19 ...

Interactive Elixir ...
iex(foo@hostname)1>

New shell named bar, but we connected to foo.
